Imagens sugeridas para a landing page:

1. Para a seção hero:
   - Uma imagem de um profissional ou equipe trabalhando com tecnologia
   - Nome sugerido: hero-image.png ou hero-image.jpg

2. Para a seção de recursos/serviços:
   - Imagens de pessoas usando diferentes tecnologias para cada serviço
   - Nomes sugeridos: service-dev.png, service-ai.png, service-automation.png

3. Para a seção de depoimentos:
   - Foto do cliente ou representante da empresa
   - Nome sugerido: testimonial-client.png

4. Para a seção CTA:
   - Imagem de um profissional sorridente com um dispositivo móvel ou notebook
   - Nome sugerido: cta-image.png

5. Logotipos de clientes/parceiros:
   - Logotipos de empresas parceiras ou clientes
   - Nomes sugeridos: client-logo-1.png, client-logo-2.png, etc.

Fontes recomendadas para imagens gratuitas:
- Unsplash (https://unsplash.com)
- Pexels (https://www.pexels.com)
- Pixabay (https://pixabay.com)

Dicas para escolher imagens:
- Escolha imagens com pessoas diversas
- Prefira imagens com tons azuis para combinar com a paleta de cores da empresa
- Busque imagens com espaço negativo para adicionar texto se necessário
- Escolha imagens de alta resolução e profissionais
